# Multi-Tenancy

Notify is fully **multi-tenant**.

Each tenant has:
- Templates
- Users & roles
- Integrations
- Notification history

---

## Tenant Identification

- JWT token OR
- Request header

---

## CSTAR Integration

CSTAR enables:
- Group management
- Role delegation
- Shared services integration

---

## Key Benefit

Each tenant operates independently with isolated configuration.
